E-commerce Testing Project (Web + API + Kafka)

## Description
This project represents testing of an e-commerce system with web interface, REST API and event-driven architecture using Kafka.

## Scope
- Functional testing
- API testing
- UI automation
- Database validation
- Event validation (Kafka)

## Tech Stack
- Python, Pytest, Playwright
- Postman
- SQL (PostgreSQL)
- Docker
- Kafka (conceptual testing)

## Tested Features
- User authentication
- Product catalog
- Cart functionality
- Order creation

## Results
- 30+ test cases created
- 15+ bugs reported
- API collection created and tested
- Basic UI automation implemented
